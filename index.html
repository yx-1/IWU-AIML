A little about me

Hi, I'm Yuxuan — a student studying AI and Data Analytics at Indiana Wesleyan University. I'm drawn to problems where data and clear communication overlap: understanding how models work, why they make certain decisions, and how to explain that to someone who doesn't have a statistics background.

This site is a living collection of my coursework. Each project here started as a class assignment that I ended up caring about — whether that's visualizing ML algorithms, exploring datasets, or building something interactive. I'll keep adding to it as the semester goes on.

A Visual Guide
Machine Learning Algorithms
Supervised Learning Learns from labeled data
Decision Tree
Tabular
e.g. Approving or rejecting bank loan applications
Asks yes/no questions like a flowchart until it reaches an answer.
Random Forest
Tabular
e.g. Predicting hospital patient readmission risk
Builds many trees on random data slices, then takes a majority vote.
Logistic Regression
Tabular
e.g. Classifying emails as spam or not spam
Outputs a 0–100% probability via an S-curve. Above 50% = one class, below = the other.
Support Vector Machine
Tabular
e.g. Classifying tumours as malignant or benign
Draws the widest possible gap between two classes to separate them clearly.
K-Nearest Neighbors
Tabular
e.g. Recommending movies based on similar users' taste
Looks at the K closest known points and takes a vote. No training needed.
Unsupervised Learning Finds patterns on its own
K-Means Clustering
Tabular
e.g. Grouping customers by purchasing behaviour
You pick K. It places centers, assigns points to nearest one, repositions. Repeats until stable.
Hierarchical Clustering
Tabular
e.g. Grouping genes with similar expression patterns
Merges the two closest points step by step into a tree. Cut at any level to choose group count.
Principal Component Analysis
Tabular
e.g. Compressing image data before feeding into a model
Finds directions of most variation and compresses data — keeping the important structure.
Deep Learning Neural networks with many layers
Convolutional Neural Network
Computer Vision
e.g. Detecting pedestrians and signs in self-driving cars
Scans images with sliding filters — finding edges, then shapes, then whole objects layer by layer.
Transformer
NLP
e.g. Powering large language models like ChatGPT
Reads all words at once, using "attention" to weigh how much each word matters to every other.
